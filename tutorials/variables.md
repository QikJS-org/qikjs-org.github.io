# Variables, Types & Operators

Variables store data. In JS, you can use `let`, `const`, or `var`.

```js
let age = 18;
const name = "Alice";
```
### Data Types

-   String: `"Hello"`
    
-   Number: `42`
    
-   Boolean: `true / false`
    
-   Null / Undefined
    

### Operators

-   Arithmetic: `+ - * / %`
    
-   Comparison: `== === != !== > <`
    
-   Logical: `&& || !`
