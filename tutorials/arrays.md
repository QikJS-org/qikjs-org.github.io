# Arrays & Objects

## Arrays
Store multiple values in a list.

```js
const fruits = ["apple", "banana", "cherry"];
fruits.push("orange");
console.log(fruits[1]); // banana
```
## Objects

```js
Store key-value pairs.

const user = {
  name: "Alice",
  age: 18
};
console.log(user.name); // Alice
```
